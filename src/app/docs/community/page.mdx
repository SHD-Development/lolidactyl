# 社群規則

為維護社群良好之交流環境，特制定本規則。所有成員均應遵守以下規定。

## 基本行為規範

### 尊重他人
- 應尊重不同觀點與意見，進行理性且具建設性之討論
- 我們基本不限制言論自由，但請盡量不要違反 Discord ToS

### 內容規範
- 不得發布違反法律法規之資訊或煽動他人從事違法行為
- 應避免發送過度負面或可能引起他人不適之內容

### 廣告與推廣
- 未經授權，不得在論壇區以外發送任何商業廣告或推廣內容
- 嚴禁發送垃圾訊息或惡意連結
- 不得濫用私人訊息功能進行商業推廣活動

## 無效簽到

### 定義

每日簽到係指於「`#》快來聊天｜💬`」頻道內發送之首則訊息。若該訊息內容為無實質意義之字詞，如：「hi」、「嗨」、「安安」等類似用語、圖片、表情符號或貼圖，則視為無效簽到。

### 有效簽到示例

以下行為不被認定為無效簽到：

1. 與其他成員進行有意義之交流互動
2. 提出疑問或尋求協助
3. 回覆其他成員之訊息內容
4. 分享具有參考價值之內容或個人經驗

### 特殊時段

每日 00:00 至 01:00 時段內發送之訊息不適用於本條款規範。

## 洗版行為

### 定義

凡於伺服器內傳送多則重複性字詞、圖片、表情符號、貼圖或其他相同內容者，均屬「洗版」行為。

### 避免洗版之建議

- 應將相關內容整合為單一訊息後發送
- 建議使用編輯功能補充內容，避免連續發送多則訊息
- 不得於短時間內重複發送相同或類似內容

## 技術支援規範

### 提問準則
- 應詳細說明問題發生之情況及相關錯誤訊息
- 需提供相關之系統資訊以利問題診斷
- 避免就相同問題進行重複提問

### 協助他人
- 應以積極友善之態度協助其他成員
- 提供資訊時應確保其準確性及可靠性
- 對於不確定之資訊應明確標示並建議尋求其他協助

## 違規處理機制

### 處罰等級制度
- 初次違規：口頭警告處理
- 輕微違規：正式警告
- 嚴重或重複違規：實施暫時禁言或移除出群組

### 嚴重違規行為
以下行為將面臨嚴厲處罰：
- 惡意攻擊伺服器系統或對其他成員進行網路攻擊
- 發布任何違反法律法規之內容
- 蓄意破壞社群秩序或進行惡意搗亂行為

## 申訴處理程序

若對處罰決定有異議，可透過以下正式管道提出申訴：
- 開客服單，詳細說明申訴事由
- 提供相關證據資料以支持申訴內容
- 耐心等待管理團隊之處理結果及回覆

## 免責聲明與附則

- SHD Cloud 對本規則擁有最終解釋權及修改權
- 本規則內容可能因應實際需要進行調整，請定期查閱最新版本
- 管理團隊有權依據實際情況對各種狀況進行彈性處理
- 本規則自公告之日起生效，所有成員均應遵守
